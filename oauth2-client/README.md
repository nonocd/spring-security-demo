# Spring Boot + OAuth 2 Password Grant - Hello World Example

以下是根据OAuth2规范的授权类型-
- 授权码模式 (authorization_code)
- Implicit grant (implicit)
- 资源所有者密码凭证模式 (password)
- 客户端凭证模式 (client_credentials)
- 刷新令牌 (refresh_token)
